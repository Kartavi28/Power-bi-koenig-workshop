# 📊 Dashboard in a Day – Power BI Project
This repository contains my work completed during the **Dashboard in a Day workshop** delivered by **Koenig Solutions – Canada**.
---

## 🚀 1.Project Overview

This project demonstrates end-to-end **Power BI development skills**, including data preparation, modeling, visualization, and business insights creation.

Key areas covered:

- 📥 Data import and transformation using **Power Query**
- 🔗 Data modeling and relationship management
- 📊 DAX measures and calculated columns
- 📈 Interactive dashboards and visual storytelling
- 🎯 KPI tracking and performance analysis
- 🔍 Drill-through, filters, and slicers
- ☁️ Dashboard publishing using **Power BI Service**
---

## 📄2. Report Page – Manufacturer Analysis

The main report page focuses on **Manufacturer performance insights**, including:

- 🌍 Revenue by Country (Column Chart)
- 📉 Revenue vs Previous Year Sales (Gauge Visual)
- 📊 Revenue & % Growth Trend (Combo Chart)
- 🧩 Manufacturer Slicer for dynamic filtering
- 🧮 Product Hierarchy Matrix
- 📈 Sales Growth KPIs and metrics

---
## 🛠️3. Tools & Technologies

- 📊 Power BI Desktop  
- 🔄 Power Query  
- 🧠 DAX (Data Analysis Expressions)  
- ☁️ Power BI Service

## 🛠️4. Data Source
 📁 Workshop dataset from **Dashboard in a Day – Koenig Solutions (Canada)**

## ✨ Features & Highlights

- 📊 Interactive Power BI dashboard for manufacturer performance analysis
- 🌍 Revenue analysis across multiple countries
- 📈 Year-over-Year (YoY) sales comparison using DAX measures
- 📉 Revenue growth tracking with KPI indicators
- 🔍 Dynamic filtering using slicers and interactive visuals
